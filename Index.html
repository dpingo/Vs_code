Visual Code Studio

Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

What is console?

In JavaScript, The console object provides access to the browser's debugging console (e.g. the Web console in Firefox).

What is the purpose of console?

Console is used to print any kind of variables defined before in it or to just print any message that needs to be displayed to the user.

How to open console screen?
